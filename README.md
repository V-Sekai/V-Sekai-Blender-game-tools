## Quickstart NXT

* First time install
    For just the cli and backend: `pip install nxt-core`
    For the backend and the visual editor: `pip install nxt-editor`
* Launch (nxt editor)
    `nxt ui`
* Update
    `pip install -U nxt-core`
    `pip install -U nxt-editor`
